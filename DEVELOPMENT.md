# Development Notes

## Project Structure

This is a Next.js project migrated from the original lume-studio-next-main repository.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- Next.js 15 with App Router
- TypeScript configuration
- Tailwind CSS for styling
- Motion.dev for animations
- Radix UI components
- Custom motion primitives
- Responsive design
- Dark/light theme support

## Components

### Motion Primitives
- InView - Scroll-triggered animations
- AnimatedGroup - Staggered animations
- TextEffect - Text animation presets
- InfiniteSlider - Continuous scrolling
- Cursor - Custom cursor animations
- MorphingDialog - Morphing dialog interactions
- SlidingNumber - Animated number counters
- ProgressiveBlur - Layered blur effects

### UI Components
- Button, Input, Card, Badge, Avatar
- Label, Select, Textarea
- Header, Footer, Navigation

### Sections
- Hero section with video background
- About, Services, Portfolio
- Stats, Testimonials, Contact
- Full version section with waitlist

## Dependencies
